# Бренды Тамбовщины

Прототип сайта «Интерактивная карта региональных брендов Тамбовской области» на Next.js, React, TypeScript и Tailwind CSS.

## Что реализовано

- главная страница с hero-блоком, шестиугольной медиакартой и популярными брендами;
- каталог брендов с поиском, фильтрами, сортировкой и пустым состоянием;
- детальная страница бренда с рейтингом, RadarChart, продукцией, точками продаж и официальными ссылками;
- mock-карта Тамбовской области с маркерами и боковой панелью;
- страницы «О проекте», «Добавить бренд», «Контакты»;
- прототип административной панели;
- сохранение заявок производителей в localStorage.

## Запуск

```bash
npm install
npm run dev
```

После запуска сайт будет доступен на `http://localhost:3000`.

На Windows, если PowerShell блокирует `npm.ps1`, используйте:

```powershell
npm.cmd install --ignore-scripts
npm.cmd run dev
```

Окно терминала с `npm.cmd run dev` нужно оставить открытым.
